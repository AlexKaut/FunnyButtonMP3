# FunnyButtonMP3
Instructions on how to make a meme button on an arduino with a mp3 module
![Video alt]([Videos/Yellow-leds-dark.mp4](https://github.com/Alex-Kaut/FunnyButtonMP3/blob/main/Videos/Yellow-leds-dark.mp4))

# Introduction
Initially, this project is designed for a "lamp in the form of a keyboard key", but, of course, this concept and schemes can be safely assembled in any other case. An important point of this project is different assembly options - you can choose the power supply and LEDs that suit you
